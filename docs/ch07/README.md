# 机械加工表面质量

- 对象

  机械零件的加工表面和表面层

- 作用

  机器的性能主要取决于零件的表面质量

- 主要内容

  - 机械加工表面质量的含义
  - 表面质量对使用性能的影响
  - 表面质量产生的机理
  - 对生产现场中发生的表面质量问题,如受力变形、磨削烧伤、裂纹和振纹等从理论上做出解释
  - 提出提高机械加工表面质量的途径

## 机械加工后的表面质量

### 表面质量的含义

- 含义的发展

  - 研究初期

    - 微观几何特征如表面粗糙度和表面微裂纹等表面外部加工效应作为衡量表面加工质量的主要依据
    - 认为表面粗糙度与零件的使用性能之间存在着直接关系

  - 实际上

    - 损坏多是从表面之下几十微米范围内开始的
    - 表面之下的**冶金物理**和**力学性能变化**对使用性能影响很大

  - 目前

    - 表面粗糙度仅是评价和控制表面加工质量的一个指标

    - 加工过程中应该使用表面完整性指标对零件表面加工质量进行综合性评价

- 零件切削加工表面完整性的主要方面

  - 与表面形貌或表面纹理组织有关的部分

    研究零件最外层表面与周围环境间界面的几何形状

    包括表面微观几何形状写表面缺陷等表面特征,属于外部加工效应,通常用表面粗糙度等来衡量

  - 与加工表面层物理力学性能状态有关的部分

    研究表面层内的特性,属于内部加工效应

    包括表面内的残余应力、加工硬化、金相组织变化、裂纹等

- 表面质量的主要内容

  - 表面层的几何形状

    - 表面粗糙度
    - 表面波纹度

      间距大于表面粗糙度但小于表面几何形状误差的表面几何不平度

  - 表面层的物理力学性能

    - 表面层冷作硬化(简称冷硬)

      表层金属冷态塑性变形后, 强度和硬度有所提高

    - 表面层金相组织的变化

      切削热引起工件表面温升

    - 表面层残余应力

    - 裂纹

      热态塑性变形和金相组织变化的影响 -> 表面层残余拉应力超过材料的强度极限 -> 表面出现裂纹

### 表面质量对零件使用性能的影响

- 对零件耐磨性的影响

  例如车削或铣削加工后的表面实际接触面积仅为名义接触面积的 15% ~20% ,精磨后可达到 30% ~50% ,只有研磨后才能达到 90% ~97%

  - ![][6]
  - 如载荷加大或润滑条件恶化,磨损曲线将向上向右移动,最佳表面粗糖度值也随之右移。
  - 当表面粗糖度值小于最佳值时,零件实际接触面积増大,接触表面之间的润滑油被挤出,金属表面直接接触,因金属分子间的亲和力而发生粘结(称为冷煤),随着相对运动的进行,粘结处在剪切力的作用下发生斯裂破坏。有时还由于摩擦产生的高温、摩擦面局部馆化(称为热燥)等,使接触表面遭到破坏,初期磨损量反而急剧增加。
  - ,还与表面粗糖度的轮腐形状及纹路方向有关
    - 耐磨性取决于轮廊峰顶形状和凹谷形状。前者决定干摩擦时的实际接触面积,后者决定润滑摩擦时的容油情况
    - ![][7]
  - 表面层的冷硬可显著地减少零件的磨损。其原因是冷硬提高了表面接触点处的屈服强度, 减少了进一步塑性变形的可能性,并减少了摩擦表面金属的冷塀现象。但如果表面硬化过度,零件心部和表面层硬度差过大,会发生表面层剥落现象,使磨损加剧。表面层产生金相组织变化时,由于改变了基体材料原来的硬度,因而也直接影响其耐磨性。

- 对零件疲劳强度的影响
  - 周期性的交变载荷作用
  - 表面粗糖度值越大,即凹陷越深和越尖,应力集中越严重,越容易形成和扩展疲劳裂纹而造成零件的疲劳损杯
  - 零件对应力集中敏感,钢材的强度越高,表面粗糖度对疲劳强度的影响越大。
  - 加工纹路方向对疲劳强度的影响更大,如果刀痕与受力方向垂直,则疲劳强度将显著降低
  - 零件表面的冷硬层能够阻碍裂纹的扩大和新裂纹的出现
  - 但冷硬层过深或过硬则容易产生裂纹,反而会降低疲劳强度
  - 表面层残余的压应力能够部分抵消工作载荷施加的拉应力,延缓疲劳裂纹扩展。而残余拉应力容易使已加工表面产生裂纹而降低疲劳强度。
- 对零件耐烛性的影响
  - 零件表面粗糖度值越大,潮湿空气和腐烛介质越容易堆积在零件表面凹处而发生化学腐烛, 或在凸峰间产生电化学作用而引起电化学腐烛,故耐烛性越差
  - 表面冷硬和金相组织变化都会产生内应力。零件在应力状态下工作时,会产生应力腐烛,若有裂纹,则更增加了应力腐烛的敏感性。因此表面内应力会降低零件的耐烛性
- 对配合质量的影响
  - 表面粗糖度影响实际配合精度和配合质量
  - 间隙配合
    - 磨损量太大时,会使配合间隙增大,以政改变原定的配合性质
    - 过盈配合
  - 过盈量减小,影响配合的可靠性
- 对零件的其它影响
  - 表面质量对零件的密封性能及摩擦系数都有很大的影响
  - 恰当的表面粗糖度值能提高滑动零件的运动灵活性,减少发热和功率损失
- 零件表面层状态对其使用性能有如此大的影响是因为承受载荷应力最大的表面层是金属的边界,机械加工后破坏了晶粒的完整性,从而降低了表面的某些力学性能。表面层有裂纹、加工痕迹等各种缺陷,在动载荷的作用下,可能引起应力集中而导致破坏。
- 机械加工后的表面粗糖度
  - 切前加工后的表面粗縫度
    - 几何因素、物理因素和工艺系统的振动
    - 几何因素
      - 由刀具相对于工件作进给运动时在加工表面上遗留下来的切削层残留面积
      - ![][8]
      - ![][9]
      - ![][10]
      - 背吃刀量和进给量很小时,表面粗植度主要由刀尖圆弧构成
        - ![][11]
    - 物理因素
      - 切前过程中刀具的刃口圆角及后面对工件挤压与摩擦而产生塑性变形,铺性越好的材料塑性变形就越大,且容易出现积屑瘤与麟刺,使表面粗糖度严重恶化。此外,还有切削用量、冷却润滑液和刀具材料等因素的影响
  - 磨削加工后的表面粗糖度
    - 与磨削过程和砂轮结构有关的几何因素、与磨削过程和被加工材料塑性变形有关的物理因素和工艺系统的振动因素
    - 几何因素看,砂轮上磨粒的微刃形状和分布对于磨削后的表面粗糖度是有影响的
    - 为了降低表面粗糖度值
      - 砂轮的粒度
        - 表面粗糖度值愈小
        - 砂轮的修整
          - 表面粗糖度值也就愈小
        - 砂轮速度
          - 刻痕数
          - 同时塑性变形造成的隆起量随着砂轮速度的提高而减少,原因是高速下塑性变形的传播速度小于磨削速度,材料来不及变形
      - 工件速度
        - 工件速度越大,单个磨粒的磨削厚度就越大,单位时间内磨削工件表面的磨粒数减少,表面粗糖度值增大
      - 径向进给量
        - 增大磨削径向进给量将增加塑性变形的程度,从而增大表面粗糖度值
        - 采用较大的径向进给量,以提高生产率
        - 小径向进给量或无径向进给量磨削,以减小表面粗糖度值
      - 轴向进给量
        - 较小的轴向进给量,则磨削后表面粗糖度值较小
      - 引起磨削表面粗糖度值增大的主要原因还往往是工艺系统的振动,增加工艺系统刚度和阻尼,做好砂输的动平衡以及合理地修整砂轮可显著降低表面粗糖度值
- 机械加工后表面层的物理力学性能

  - 机械加工后表面层的冷作硬化
    - 层的冷作硬化 1.
      - 表面层金属由于塑性变形使凰体间产生剪切滑移,晶格发生拉长、扭曲
      - 和破碎而得到强化
      - 特点是变形抵抗力提高(屈服强度提高),塑性降低(相对伸长率降低)
      - 冷作硬化的指标通常用冷硬层的深度 A、表面层的显微硬度 H 以及硬化程度 N 来表示(圏 7.8),其中 N=H/HO ,仏为原来的显微硬度
      - ![][12]
      - 表面层冷作硬化的程度决定于产生塑性变形的力、变形速度及变形时的温度。力越大,塑性变形越大,则硬化程度越大;速度越高,塑性变形越不充分,则硬化程度越小;变形时的温度不仅影响塑性变形程度,还会影响变形后金相组织的恢复程度。
      - 完全强化。
        - 出现晶格歪扭以及纤维结构和变形层物理、力学性能的改变
      - 不完全强化
        - 若温度超过(0.25~0.30)Tg(雄化绝对温度),则除了强化现象外,同时还有回复现象,此时歪扭的晶格局部得到恢复,减小了冷硬作用;如果温度超过 0.4Ta ,就会发生金属再结晶,此时由于强化而改变了的表面层物理、力学性能几乎可以完全恢复
      - 冷作硬化就是强化作用和回复作用的综合结果
      - 切削温度越高、高温持续时间越长、强化程度越大，则回复作用也就越强。因此对高温下工作的零件，能保证疲劳强度的最佳表面层是没有冷硬层或者只有极小(10~20m)冷作硬化的表面层
    - 影响冷作硬化的主要因素
      - 刀具。刀具的切削刃口圆角和后面的磨损量对于冷硬层有很大的影响,此两值增大时, 冷硬层深度和硬度也随之增大。前角减小时,冷硬层深度也增大。
      - 切削用量。
        - 切削速度提高时,刀具与工件接触时间短,塑性变形程度减小,同时会使温度升高,有助于冷硬的回复,所以硬化层深度和硬度都有所减小
        - 进给量增大时,切削力增大,塑性变形程度也增大,因此硬化现象增大。但在进给量较小时,由于刀具的刃口圆角在加工表面单位长度上的挤压次数增加,因此硬化倾向也会增大。径向进给量增大时,冷硬层深度也有所增大, 但其影响不显著。
      - 被加工材料
        - 硬度愈低、塑性愈大,切削后的冷硬现象愈严重
  - 机械加工后表面展金相组织的变化
    - 金相组织变化的原因
      - 磨削区温度可上升到 400 ~1 000 力,甚至更高。这样高的加热速度促使加工表面局部形成瞬时热聚集现象,有很高温升和很大的温度梯度,出现金相组织的变化,强度和硬度下降,产生残余应力,甚至引起裂纹,这就是磨削烧伤现象
      - 磨削洋火钢时表面层产生的烧伤有以下三种:
        - 回火烧伤
          - 温度超过马氏体转变温度而未超过相变温度,则工件表面原来的马氏体组织将产生回火现象,转化成硬度降低的回火组织,即索氏体或屈氏体
        - 粹火烧伤
          - 温度超过相变温度,马氏体转变为奧氏体,由于冷却液的急冷作用,表层会出现二次洋火马氏体,硬度较原来的回火马氏体高,而它的下层则因为冷却缓慢而成为硬度降低的回火组织。
        - 退火烧伤
          - 不用冷却液进行干磨削时,磨削区温度超过相变温度,马氏体转变为奥氏体, 因工件冷却缓慢,则表层硬度急剧下降,这时工件表层被退火
    - 影响磨削加工时金相组织变化的因素
      - 工件材料、磨削温度、温度梯度及冷却速度
      - 低碳钢时不会发生相变
      - 高合金钢如轴承钢、高速钢、镇絡钢等传热性特别差,在冷却不充分时易出现磨削烧伤;未洋火钢为扩散度低的珠光体,磨削时间短时不会发生金相组织的变化; 粹火钢极易相变。
      - ![][13]
        - 当磨削深度小于 10 |xm 时,由于温度的影响使表面层的回火马氏体产生弱化,井与塑性变形产生的冷作硬化现象
        - 综合而产生比基体硬度低的部分,而表面的里层由于磨削加工中的冷作硬化起了主导作用而又产生比基体硬度高的部分。当磨削深度为 20 ~30 &m 时,冷作硬化的影响减小,磨削温度起主导作用。由于磨削区温度高于马氏体转变温度、低于相变温度而使表面层马氏体回火产生回火烧伤。当磨削深度增大至 50 &m 时,磨削区最高湿度超过相变临界温度,急冷时产生洋火烧伤, 而再往里层则硬度又遂渐升高直至未受热影响的基体组织。
  - 机械加工后表面展的残余应力

    - 残余应力产生的原因

      - 冷态塑性变形
        - 机械加工时,表层金属产生强烈的塑性变形
        - 沿切削速度方向表面产生拉伸变形,晶粒被拉长,金属密度会下降,即比体积增大,而里层材料则眶碍这种变形,因而在表面层产生残余压应力,在里层则产生残余拉应力。
      - 热态塑性变形
        - ![][14]
        - 切削或磨削热使工件表面局部温升过高,引起高温塑性变形
        - 第 1 层温度在塑性温度以上,这层金属产生热塑性变形;第 2 层温度在塑性温度与
        - 室温之间,这层金属只产生弹性热膨胀;第 3 层处在室温的冷态层,不产生热变形。由于第 1 层处于塑性状态,故没有应力;第 2 层的膨胀受到第 3 层的阻碍,所以第 2 层产生压应力;第 3 层则产生拉应力(图 7.lOb)。 开始冷却时,第 1 层冷到塑性温度以下,体积收缩,但第 2 层阻碍其收缩,这时第 1 层中产生拉应力,第 2 层中的压应力增加。而由于第 2 层的冷却收缩,第 3 层中的拉应力有所减小(圉 7.10c)。最后冷却时,第 1 层继续收缩,拉应力进一步增大,而第 2 层热膨胀全部消失,完全由第 1 层的收缩而形成一个不大的压应力,第 3 层拉应力消失,而与第 2 层一起受第 1 层的影响,也形成一个不大的压应力(图 7. lod)。
      - 金相组织变化
        - 高温会引起表面的相变
        - 不同的金相组织有不同的比体积,表面层金相变化的结果将造成体积的变化。表面层体积膨胀时,因为受到基体的限制, 产生了压应力;反之,表面层体积缩小时,则产生拉应力
      - 在一定条件下，其中某一种或两种因素可能起主导作用。  
         例如，切削时若切削热不多则以冷态塑性变形为主，若切削热多则以热态塑性变形为主。磨削时表面层残余应力随磨削条件的不同而不同，图 7.11 所示为三类磨削条件下产生的表面层残余应 edw/tf 中等窖削力。轻磨削条件产生浅而小的残余压应力，因为此时没有金相组织变化，温度影响也很小，主要是塑性变形的影响在起作用；中等磨削条件产生浅而大的拉应力；淬火钢重磨削条件则产生深而大-157 人轻削的拉应力（最外表面可能出现小而浅的压应力）， 274 这里显然是由于热态塑性变形和金相组织变化的影响在起主要作用的缘故。

    - 磨削裂纹的产生
      - 实碌机械加工后的表面层残余应力及其分布,是上述三方面因素综合作用的结果
      - 磨削加工中热态塑性变形和金相组织变化的影响较大
      - 有残余拉应力
      - 裂纹总是拉应力引起的,且常与烧伤同时出现
      - 当残余拉应力超过材料的强度极限时,零件表面就会出现裂纹
